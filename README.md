### Dependencies
The following dependencies will be installed by the install script if not already installed:
- conky-all
- ffmpeg
- net-tools

### Installation
```sh
cd wetch
./install

```

### Run
```sh
./wetch
```

### Kill
Kill by calling
```sh
pkill -f wetch
```
